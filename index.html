LESSON 1 - The Basics of HTML

HTML // hypertext markup language

html is what forms the web. It contains text (what you see) and markup (what determines how it looks) html is formed by codes and elements which will tell the browser how to read a page.

tags are how a computer and a browser will read information the way we want it. Each tag means something else and is always enclosed by the tag signs. Most of the times we must place a tag at the beginning and close it at the end of the element you want to affect.

some examples of tags are:
b = bold em = italic a = anchors or links br = break p = paragraph

LESSON 3 - CSS style sheets

AVOID REPETITION// simplifying style

Avoiding repetition is important in order to reduce the probability of errors when writing code, as well as when making changes in the future. CSS is used as a way to simplify the styling of a web page by using divs and classes and writing specific characteristics we want for each of those (font style, size, location, color, etc). CSS stands for Cascading Style Sheets and lets us avoid repetition. We can give a certain style to a certain class and once we link our CSS to the HTML, the changes will be applied to all elements that carry that same class name.

In this particular website, for example, I gave the same style to every LESSON by assigning each lesson a same div and class. I then specify the style I wanted for that particular div and class on my CSS and that did the trick! If I would copy/paste the style attribute for each lesson instead, I would have more margin of error and more possibilities of making mistakes when adding and changing elements.

LINKING CSS// different ways

Although there are different ways to use CSS on an HTML, it is recommended to write the CSS code on a sparate document, save it on the same folder as the HTML and then link them together with a specific line of code. This method will avoid confusion, and it is cleaner for others to read and understand.

LESSON 4 - Programming

PROGRAMMING // languages and interpreters

The computer is a machine that can be programmed to do almost anything. There are different languages that can be used to tell a computer what do to. Unlike english, spanish or any other laguages we can learn, programming languages are more precise. Computers use interpreters to understand the laguage, but interpreters can't distinguish "incorrect" code, we must write precise and correct language in order for the code to run properly.

PYTHON // grammar and expressions

In this lesson, we got to understand a bit one of the programming languages called PYTHON and Python Interpreter which is used to follow the instructions on the code we write. Python Grammar is based on formulas formed by series of what we call expressions

An example of a Python grammar is: expression--->Expression Operator Expression

In this example, we can write a combination of one expression followed by one operator and another expression. Expressions and Operators can be replaced by different elements such as:

Expression--->numbers

Operator--->+

Operator--->*

Number--->0,1,2, ...

Taking this example we could write something like: print 2 * 5 + 3 * 6
